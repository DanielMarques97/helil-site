# HELIL Site

Landing page desenvolvida para a Associação HELIL de Amparo à Criança, com o objetivo de apoiar a divulgação das oficinas culturais, esportivas e educacionais da instituição.

## 🔗 Site publicado

https://danielmarques97.github.io/helil-site/?v=999

## 🎯 Objetivo do projeto

Criar uma página institucional simples, responsiva e acessível para apresentar a HELIL, suas oficinas, localização e canais de contato.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## 📌 Funcionalidades

- Seção institucional sobre a HELIL
- Carrossel de oficinas
- Mapa com localização
- Botão de contato via WhatsApp
- Layout responsivo para celular e computador

## 🎓 Contexto acadêmico

Projeto desenvolvido como parte do Projeto de Extensão II do curso de Análise e Desenvolvimento de Sistemas, com foco em ação e difusão cultural por meio da tecnologia.
