# HELIL Site - versão refatorada

Estrutura sugerida:

helil-site/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── imagens/
    ├── futebol/
    ├── musica/
    ├── luta/
    └── artes/

Esta refatoração tem o intuito de separar a folha de estilos CSS e a lógica JavaScript para implementações futuras sem depender de um único arquivo index.html.
